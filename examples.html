Text Summarization

Summarize a complex topic into simple terms.

Explain the concept of blockchain technology as if you were talking to a complete beginner. Use an analogy to make it easier to understand. Avoid technical jargon. The summary should be no more than three paragraphs.

Expected AI Output

Imagine a shared digital notebook that everyone can see but no one can erase. Every time someone adds a new page (a "block"), it's magically linked to the previous page, creating a chain. This notebook isn't stored in one place; instead, a copy is given to everyone in the group.

Because everyone has a copy, it's incredibly secure. If someone tries to secretly change a page in their notebook, it won't match everyone else's, and their change will be rejected. This makes it almost impossible to cheat or alter the records.

This shared, secure notebook is the basic idea behind blockchain. It's a way to record transactions or information in a transparent and tamper-proof way, without needing a central authority like a bank to verify everything.

Code Generation

Generate a JavaScript function for a common web task.

Write a JavaScript function that takes an email address as a string and returns `true` if the email is valid and `false` otherwise. A valid email should contain one "@" symbol and at least one "." after the "@". Include comments explaining the regex used.

Expected AI Output

/**
 * Validates an email address using a regular expression.
 * @param {string} email The email address to validate.
 * @returns {boolean} True if the email is valid, false otherwise.
 */
function isValidEmail(email) {
  // Regular expression for basic email validation.
  // It checks for a pattern of: [characters]@[characters].[characters]
  const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  
  // Test the email against the regex and return the result.
  return emailRegex.test(email);
}

Chain-of-Thought: Logic Puzzle

Guide the AI to solve a multi-step logic problem.

Q: I have a 5-gallon jug and a 3-gallon jug. How can I measure out exactly 4 gallons of water?

A: Let's think step by step to solve this classic water jug puzzle.
1. Fill the 5-gallon jug completely.
2. Pour water from the 5-gallon jug into the 3-gallon jug until it's full. Now the 5-gallon jug has 2 gallons left.
3. Empty the 3-gallon jug.
4. Pour the 2 gallons from the 5-gallon jug into the 3-gallon jug.
5. Fill the 5-gallon jug completely again.
6. Carefully pour water from the 5-gallon jug into the 3-gallon jug (which already has 2 gallons) until it's full. This will take exactly 1 gallon.
7. The 5-gallon jug now has exactly 4 gallons left.
The answer is to follow these steps.

Expected AI Output

Here's a concise summary of the steps:

    1. Fill 5-gallon jug.
    2. Pour from 5-gallon to 3-gallon (5 has 2 gallons left).
    3. Empty 3-gallon jug.
    4. Pour 2 gallons from 5-gallon to 3-gallon.
    5. Fill 5-gallon jug.
    6. Pour from 5-gallon to 3-gallon (until 3-gallon is full - uses 1 gallon from 5-gallon).
    7. 5-gallon jug now contains 4 gallons.
